HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membangun struktur halaman web. HTML berfungsi sebagai kerangka utama sebelum styling dan interaksi ditambahkan.
Pada langkah ini, Anda mempelajari struktur dasar dokumen HTML yang terdiri dari deklarasi <!DOCTYPE html>, elemen <html>, <head>, dan <body>. Struktur ini wajib ada agar browser dapat membaca dokumen dengan benar.
Selanjutnya, Anda diperkenalkan dengan tag HTML penting seperti heading, paragraf, link, dan gambar. Tag-tag ini digunakan untuk membagi dan mengatur konten di dalam halaman web.
Sebagai latihan awal, Anda akan membuat file HTML pertama dengan ekstensi .html, menuliskan struktur dasar HTML, lalu membukanya langsung di browser untuk melihat hasilnya.
